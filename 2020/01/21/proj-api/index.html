基本知识

PROJ是什么？

PROJ 是安装GMT时依赖的一个程序，功能是提供地理坐标、大地测量坐标和投影坐标的转换功能。GMT中的投影、坐标转换和椭球面距离计算等功能来自PROJ。如果用户自己加入或者修改投影，则修改此程序。
PROJ项目开始于1983年。

PROJ怎么安装

PROJ使用c++编写，从官网下载源程序，依照编译指南编译。默认安装路径为/usr/local。
如果通过在线安装，目录为/usr/bin。
